# Cosmic Cyberpunk Portfolio Asset Plan

This document lists visual assets that fit the current portfolio direction: a cosmic, cyberpunk, animated personal website with a cute futuristic cat mascot.

The goal is to make the site feel like one connected interactive universe instead of separate sections with unrelated decoration.

## Core Direction

Theme:

- Cosmic portfolio world
- Cyberpunk interface language
- Cute anthropomorphic cat mascot
- Holographic UI
- Scroll-driven movement
- Project worlds, mission logs, relay consoles, and orbital navigation

Asset style:

- Cute but polished
- Futuristic but not too dark
- Rounded shapes mixed with holographic tech
- Cyan, magenta, violet, amber, and graphite accents
- Soft glow, scanlines, orbit rings, data panels
- Website-ready silhouettes that animate clearly

## Core Mascot Assets

Create these first because the mascot can become the emotional anchor of the site.

- Cute cyberpunk cat mascot idle pose
- Cat waving pose
- Cat typing / coding pose
- Cat scanning pose
- Cat sleeping or standby pose
- Cat excited / success pose
- Cat wearing a space visor
- Cat in tiny pilot jacket
- Cat hologram projection
- Cat inside spaceship cockpit
- Tiny cat drone companion
- Mascot loading animation

Recommended use:

- Hero assistant
- Preloader
- Empty states
- Contact section
- Project modal guide
- Small interactive companion beside scroll progress or navigation

## Hero Assets

Use the hero as the first “world scene.”

- Small personal spaceship / portfolio vessel
- Floating data rings around the hero text
- Distant cyber-city planet
- Starfield parallax layers
- Nebula cloud layers
- Holographic constellation map
- Orbiting satellite silhouette
- Warp-speed particles
- Cockpit HUD overlay
- Floating mission badge

Recommended animation:

- Spaceship subtly drifts with cursor/parallax
- Stars move at different speeds
- Data rings rotate slowly
- Hero content zooms/fades on scroll exit
- Constellation lines draw in during intro

## About Section Assets

Frame About as an “Origin World” or identity dossier.

- Holographic ID card
- Origin planet
- Biometric scan rings
- Avatar scan frame
- Floating memory fragments
- Dossier tabs
- Soft scan beam
- Trait matrix chips
- Personal timeline capsule

Recommended animation:

- Section title reveals by words
- Avatar frame scans from top to bottom
- Dossier rows decrypt one by one
- Trait chips pop in with stagger
- Origin planet rotates slowly in the background

## Skills Section Assets

Frame Skills as a forge, lab, or orbital skill system.

- Cyber forge station
- Central energy core
- Orbiting skill nodes
- Mechanical arms assembling icons
- Energy crystals for skill categories
- Floating tool modules
- Code hologram fragments
- Circuit constellation lines
- Mini drones carrying icons

Recommended animation:

- Skill nodes orbit or assemble on scroll
- Central core brightens as the section enters
- Icons pop in with stagger
- Category groups rotate at different speeds
- Hovering a skill triggers glow or scan effect

## Experience Section Assets

Frame Experience as “Mission Record” or “Flight Log.”

- Mission log terminal
- Flight recorder device
- Orbital timeline path
- Satellite relay nodes
- Decrypted document panels
- Space beacon for each role
- Status stamps: complete, standby, active
- Chronological coordinate markers

Recommended animation:

- Timeline path draws vertically
- Each mission record clips open
- Record content decrypts line by line
- Active/standby record pulses softly
- Hovering a record lights a matching beacon in the background scene

- Project world thumbnails
- Wormhole gate
- Radar scan overlay
- Floating project dossier screens
- Mini spacecraft traveling between project panels

- Clearance badges

Recommended animation:

- Horizontal pinned scroll
- Project cards move like world dossiers
- Images parallax inside their frames

## Contact Section Assets

Frame Contact as a “Relay Console.”

- Communication satellite dish
- Signal wave rings
- Transmission console
- Message capsule
- Holographic input fields
- Relay antenna
- Signal beam traveling into space
- Cute mascot receiving/sending message

Recommended animation:

- Form fields boot up with stagger
- Signal rings pulse behind the submit button
- Submit triggers a beam burst
- Satellite dish rotates toward the signal
- Confirmation state shows mascot success pose

## Transition Assets

These assets help the site feel cinematic between sections.

- Wormhole tunnel
- Hyperspace streaks
- Holographic grid dissolve
- Screen-glitch scanline wipe
- Planet eclipse wipe
- Starburst flash
- Black-hole pull transition
- Data stream curtain
- Portal ring expansion
- Dither/noise dissolve

Recommended use:

- Hero to About: constellation / scanline reveal

- Skills to Projects: forge core becomes portal
- Projects modal: black-hole or wormhole transition

- Krita

For 3D assets:

- Blender
- Spline

For website integration:

- React Three Fiber for `.glb` assets
- GSAP for scroll-driven transitions

- CSS masks and clip-path for holographic reveals
- Canvas or WebGL for heavier effects

## Design Rule

Every asset should have a job.

Good asset:

- Supports the section story
- Has a clear silhouette
- Can animate simply
- Fits the cyberpunk cosmic palette
- Helps users understand the page flow

Bad asset:

- Looks cool but has no role
- Makes the page visually noisy
- Competes with text readability
- Uses a different art style from the mascot
- Slows the site without improving the experience
